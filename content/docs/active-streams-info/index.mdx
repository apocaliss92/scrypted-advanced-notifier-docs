# Active Streams Info

Scrypted plugin for checking currently active streams. Useful for monitoring stream usage and debugging.

## Features

- **Stream tracking** — See which streams are currently active
- **Settings** — Configure tracking behavior

## Installation

1. Install **Active Streams Info** from the Scrypted plugin store
2. The plugin exposes stream information via its interface

## Use Cases

- Monitor how many cameras are streaming
- Debug stream leaks or stuck connections
- Capacity planning
