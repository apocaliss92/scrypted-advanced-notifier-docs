# Scrypted Syslog

Scrypted plugin to send logs to remote syslog servers. Centralize logging for monitoring and debugging.

## Features

- **Remote logging** — Forward logs to syslog servers
- **Settings** — Configure host, port, facility, severity

## Installation

1. Install **Scrypted Syslog** from the Scrypted plugin store
2. Configure syslog server in settings

## Configuration

| Setting | Description |
|---------|-------------|
| Syslog host | IP or hostname of syslog server |
| Port | Syslog port (default 514) |
| Facility | Syslog facility code |
| Protocol | UDP or TCP |

## Use Cases

- Centralized log aggregation
- SIEM integration
- Debugging distributed setups
- Compliance and audit trails
