<PluginTitle slug="ntfy" title="Ntfy" />

[ntfy](https://ntfy.sh) integration for Scrypted. Send push notifications to your devices via ntfy.sh or a self-hosted ntfy server.

## Features

- **DeviceProvider** — Create notifier devices
- **DeviceCreator** — Add devices with topic configuration

## Installation

1. Install **Ntfy Plugin** from the Scrypted plugin store
2. Create a device and configure your ntfy topic

## Configuration

| Setting | Description |
|---------|-------------|
| Topic | ntfy topic (e.g. `my-scrypted-alerts`) |
| Server | ntfy server URL (default: ntfy.sh) |

## Self-Hosted

Use your own ntfy server URL for privacy and reliability.
