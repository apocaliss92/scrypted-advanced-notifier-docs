# Credentials Setup

The Wyze Native plugin requires two sets of credentials:

1. **Wyze Account** — email + password
2. **Wyze Developer API** — API Key + Key ID

## 1. Wyze Account Authentication

> [!IMPORTANT] Email + Password Required
> The plugin uses the Wyze REST API for device discovery, which requires **email + password** authentication. Google/Apple/Facebook sign-in is **not supported** by the API.

### If you signed up with Google/Apple/Facebook

You need to add a password to your existing Wyze account:

1. Open the **Wyze app** on your phone
2. Go to **Account** → **Security** → **Password**
3. Set a new password
4. You can now use your email + this new password with the plugin

Alternatively:
1. Go to [wyze.com](https://www.wyze.com)
2. Sign in with your Google/Apple/Facebook account
3. Navigate to **Account Settings** → set a password
4. Use your email + this password in the plugin

### If you have MFA (Two-Factor Authentication)

MFA is **not currently supported**. If you have MFA enabled, you'll need to temporarily disable it or create a secondary Wyze account and share your cameras with it.

## 2. Wyze Developer API Credentials

The API Key and Key ID are required to authenticate with the Wyze cloud API.

### How to get them

1. Go to the [Wyze Developer Portal](https://support.wyze.com/hc/en-us/articles/16129834216731)
2. Follow the instructions to request API access
3. Once approved, you'll receive:
   - **API Key** — a long alphanumeric string (e.g., `w0GNY0vMjU9E1Q...`)
   - **Key ID** — a UUID (e.g., `5f3fa74d-516e-4786-919d-b2187b516f0a`)

> [!TIP]
> The API Key and Key ID are free and typically approved within a few hours. They are tied to your Wyze developer account, not individual cameras.

## 3. Configure in Scrypted

Once you have all credentials, enter them in the plugin settings:

| Setting | Value |
|---------|-------|
| **Email** | Your Wyze account email |
| **Password** | Your Wyze account password |
| **API Key** | From the Developer Portal |
| **Key ID** | From the Developer Portal |

After saving, the plugin will automatically discover your cameras.

## Security Notes

- Credentials are stored in Scrypted's local storage (encrypted at rest)
- The Wyze cloud API is used **only for device discovery** — to retrieve P2P connection parameters (UID, ENR, MAC, IP)
- **Video streaming is 100% local** — data flows directly between the camera and Scrypted over your LAN
- No video data ever passes through Wyze's cloud servers
