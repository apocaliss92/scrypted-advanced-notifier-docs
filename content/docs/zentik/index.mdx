Zentik integration for Scrypted. Send push notifications through [Zentik](https://zentik.io). Full-featured app with rich notifications, history, and custom actions — recommended for use with [Advanced Notifier](/docs/advanced-notifier/getting-started).

## Features

- **DeviceProvider** — Create notifier devices
- **DeviceCreator** — Add devices with access token and bucket ID

## Scrypted Plugin

- **NPM:** [@apocaliss92/scrypted-zentik](https://www.npmjs.com/package/@apocaliss92/scrypted-zentik)

Install from Scrypted: **Plugins** → search **Zentik** → **Install**.

## Setup

1. Login on the Zentik app (or create an account)
2. Open **Settings → Access token** and create a new token
3. Go to **Buckets** and copy the bucket ID
4. In Scrypted, add a Zentik device with token and bucket ID
5. Configure per-camera settings if needed (Advanced Notifier → Notifier on each camera)

:::info Bucket Access
The shared bucket must have **write access** (or you must be the owner).
:::

## Advanced Configuration

Texts and translations can be customized in the notifier settings.
