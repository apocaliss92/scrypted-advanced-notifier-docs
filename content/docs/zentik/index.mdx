# Zentik Plugin

Zentik integration for Scrypted. Send push notifications through [Zentik](https://zentik.io).

## Features

- **DeviceProvider** — Create notifier devices
- **DeviceCreator** — Add devices with access token and bucket ID

## Installation

1. Install **Zentik Plugin** from the Scrypted plugin store
2. Create a device with your Zentik credentials

## Setup

1. Login on the Zentik app
2. Open **Settings → Access token**
3. Create a new token and copy it
4. Go to **Buckets** and copy the bucket ID from the provided button
5. In Scrypted, add a Zentik device and enter the token and bucket ID

:::info Bucket Access
The shared bucket must have **write access** (or you must be the owner).
:::
