# Frigate Bridge

Import object, audio, and motion events from [Frigate](https://frigate.video) into Scrypted. Videoclips, birdseye, and seamless camera adoption.

## Features

- **Frigate Motion Detection** — Starts motion on Scrypted when detected on Frigate
- **Frigate Videoclips** — Shows videoclips from Frigate on the NVR app
- **Frigate Object Detector** — Forwards object detection to Scrypted; zones configurable
- **Frigate Audio Detector** — Ingests audio levels and events (scream, yelling, barking, crying) for Advanced Notifier rules
- **Frigate Events Recorder** — Shows Frigate detections on the Scrypted NVR app (use with Object Detector)
- **Camera Adoption** — Seamless adoption of Frigate cameras with all extensions enabled
