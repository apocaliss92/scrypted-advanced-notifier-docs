# Occupancy Rules

Occupancy rules monitor a specific area to mark it as occupied or not.

## Setup

1. Set an **Object Detector** on the plugin page (Rules → Object Detector)
2. Add a new text in **Occupancy rules** and save
3. Set activation type and notifiers
4. Set detection class to monitor
5. Set camera zone (must be **Observe** type in Object detection section)

## Options

| Setting | Description |
|---------|-------------|
| Zone type | **Intersect** — object in any portion of zone. **Contain** — object fully inside (Scrypted zones only) |
| Score threshold | For static detections, keep low (Scrypted zones only) |
| Occupancy confirmation | Seconds to avoid false results |
| Force update in seconds | Force check if no detection. Any detection checks all rules |
| Zone occupied text / Zone not occupied text | Notification texts |
| Confirm occupancy with AI | Reduce false positives. Customize prompt in plugin AI section |
| Capture zone | Reduce frame for detection; may increase success rate (Show more, Scrypted only) |
| Max objects | Zone occupied if detected objects ≥ this number (Show more) |
| On-generated sequences | Sequences when clip/image ready (Show more) |
