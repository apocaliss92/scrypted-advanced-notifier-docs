# Audio Rules

Audio rules monitor the audio received by the camera. **They activate only if an audio measurement source is active.** The plugin provides an onboard audio analyser that activates when any audio rule is running.

## Setup

1. Add a new text in **Audio rules** and save
2. Set notifiers
3. Set notification text and decibel threshold

## Options

| Setting | Description |
|---------|-------------|
| Decibel threshold | Audio level to alert |
| Duration in seconds | Minimum duration to trigger (leave blank to notify immediately) |
