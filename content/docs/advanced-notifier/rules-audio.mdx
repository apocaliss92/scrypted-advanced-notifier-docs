# Audio Rules

Audio rules monitor the audio received by the camera. **They activate only if an audio measurement source is active.** The plugin provides an onboard audio analyser that activates when any audio rule is running.

## Setup

1. Add a new text in **Audio rules** and save
2. Set notifiers
3. Set notification text and decibel threshold

## Options

| Setting | Description |
|---------|-------------|
| Decibel threshold | Audio level to trigger the notification |
| Samples window | Seconds to analyze (basic object detector emits every ~2 seconds) |
| Hits percentage | Share of samples that must exceed threshold to trigger |
| Minimum notification delay | Minimum seconds between notifications |
