# Timelapse Rules

Define a timeframe; the plugin collects frames and generates a clip at the end. Timelapses appear as videoclips in the NVR app if **Enable Camera** is enabled on the plugin page.

## Setup

1. Add a new text in **Timelapse rules** and save
2. Define week days and start/end times (e.g. 11pm–8am)
3. Set notifiers for the generated clip

## Options

| Setting | Description |
|---------|-------------|
| Notification message | Message for the notification |
| Force snapshot seconds | Force a frame acquisition on a regular basis |
| Day | Week days to run (empty = all days) |
| Start time / End time | Time window for frame collection |
| Frames acquisition delay | Minimum seconds between frames (Show more) |
| Timelapse framerate | Output video framerate (Show more) |
| Generate now | Generate clip from stored frames (Show more) |
| Cleanup data | Delete stored frames (Show more) |
| On-generated sequences | Sequences to run when clip is ready (Show more) |

If Home Assistant notifier is used and the clip is &lt;50MB, it will show as push notification preview.
