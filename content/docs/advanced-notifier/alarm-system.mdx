# Advanced Alarm System

The plugin provides a security system hooked into detection rules.

## Setup

1. Create one or more detection rules with activation type **AdvancedSecuritySystem**
2. Set one or more modes to activate each rule
3. Configure the **Advanced security system** device (texts, bypass devices, etc.)

## Features

- Discovered on MQTT
- Fully compatible with HomeKit
