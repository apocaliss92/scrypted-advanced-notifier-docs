Advanced Notifier lets you easily integrate with external systems such as [Frigate](/docs/frigate-bridge) or internal ones like NVR, so you can unify multiple systems in one place.

**Devices used with Advanced Notifier must be extended with the Advanced Notifier extension. Supported types: Cameras, Sensors, and Notifiers.**

## Features

- **Home Assistant Integration** — Exposes camera metadata and detection state to Home Assistant via MQTT or WebSocket transport with full autodiscovery
- **Notification rules** — Detection, occupancy, timelapse, audio, and recording rules with customizable notifiers
- **Webhooks** — Exposes webhooks with relevant events, rule artifacts, and latest images per detection type per camera
- **Advanced alarm system** — Security system tied to detection rules, Home Assistant, and HomeKit
- **Sequences** — Define action sequences (PTZ, scripts, switches, lock/entry) executable via rules in various conditions
- **MP4 / GIF generation** — Generates video clips and GIFs for notifications and rule artifacts

The plugin also serves as a hook for an **iOS / PWA app** for live view and events, and acts as a bridge for the [Frigate Bridge](/docs/frigate-bridge) plugin. The app is available as a PWA served by Scrypted itself; iOS beta via [TestFlight](https://testflight.apple.com/join/CYnbJFqb).
